# Language Specification

## 1. Scope

本仕様は、pmlint が解析対象とする Markdown の構文制約を定義する。

## 2. Supported Markdown Elements

- 対象とする見出しレベル: `##`（H2）のみをセクション境界とする
- セクションと見なす条件: `##` で始まる行をセクション開始とし、次の `##` または EOF までをそのセクションの本文とする
- ネスト見出しの扱い: `###` 以下はセクション本文の一部とし、セクション境界としない
- コードブロックの扱い: コードブロック内の `##` はセクション境界としない
- リストの扱い: リスト項目はセクション本文の一部とする

## 3. Section Recognition Rules

- セクション名の正規化: 前後の空白をトリムする
- 大文字小文字の扱い: 区別する（`Overview` と `overview` は別セクション）
- 重複セクションの扱い: 同名セクションが複数ある場合、最初の出現のみを有効とし、残りは `duplicate-section` として報告する
